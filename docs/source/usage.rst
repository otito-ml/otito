Usage
=====

.. _installation:

Installation
------------

To use Otito, first install it using pip:

.. code-block:: console

   pip install otito

.. _quickstart:

Quickstart
------------

To use Otito, first install it using pip:

.. code-block:: console

   from otito.metrics.numpy import Accuracy
   print(Accuracy([1,0,0], [1,0,1]))
   >>> 0.6666
